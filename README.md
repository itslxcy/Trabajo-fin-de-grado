# Sistemas alternativos de comunicación para personas con discapacidad
Repositorio para la realización y seguimiento del TFG en ingeniería de la salud.
Se realizará un estudio de evaluación técnica de los sistemas alternativos de comunicación existentes para personas con discapacidad. A partir del análisis tecnológico y de las necesidades detectadas, se plantearán mejoras desde un enfoque de diseño inclusivo.
