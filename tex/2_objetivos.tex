\capitulo{2}{Objetivos}

El objetivo general de este Trabajo Fin de Grado es realizar un análisis comparativo de los sistemas aumentativos y alternativos de comunicación existentes y emplear dicha información para desarrollar una herramienta web basada en cuestionario que permita recomendar de forma personalizada el sistema más adecuado a cada usuario.

Para alcanzar este objetivo general, se plantean los siguientes objetivos específicos:

\begin{itemize}
    \item Realizar un análisis comparativo y sistemático de los sistemas aumentativos y alternativos de comunicación disponibles, considerando sus características técnicas, funcionales y de accesibilidad.
    
    \item Identificar los parámetros relevantes que influyen en la selección de un SAAC.
    
    \item Definir un modelo de recomendación basado en criterios técnicos que relacione las características del usuario con las soluciones más adecuadas.
    
    \item Desarrollar un prototipo funcional de aplicación web que implemente un cuestionario interactivo capaz de filtrar opciones y generar recomendaciones personalizadas a partir de la base de datos diseñada.
    
    \item Detectar posibles carencias o limitaciones en las soluciones actuales a partir del análisis realizado y plantear propuestas de mejora o líneas de desarrollo futuras.
\end{itemize}

Asimismo, se establecen los siguientes objetivos técnicos derivados del desarrollo del proyecto:

\begin{itemize}
    \item Diseñar la arquitectura software del sistema.
    \item Implementar la lógica de recomendación y filtrado de resultados.
    \item Desarrollar una interfaz accesible, clara e intuitiva orientada a usuarios no especializados.
    \item Implementar una base de datos estructurada de dispositivos y características.
\end{itemize}

Finalmente, desde el punto de vista formativo, el trabajo persigue los siguientes objetivos de aprendizaje:

\begin{itemize}
    \item Aplicar conocimientos de ingeniería de la salud y desarrollo software a un problema real del ámbito sanitario.
    \item Desarrollar competencias en análisis de requisitos y desarrollo web.
    \item Integrar criterios clínicos y tecnológicos dentro de un mismo proyecto interdisciplinar orientado a la toma de decisiones.
\end{itemize}
