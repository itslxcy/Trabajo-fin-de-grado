¿Qué es este recomendador?

Esta es una página diseñada como Trabajo de Fin de Grado de Ingeniería de la salud.

En la parte superior encontrará varias pestañas con descripciones acerca del ELA y los SAAC. Dirijase a cualquiera de ellas para más información.

A su vez, puede dirigirse a la pestaña de Cuestionario, introducir los datos del usuario y obtener una recomendación personalizada sobre el SAAC que mejor se ajusta a su situación.

Criterios de evaluación para el cuestionario

Al realizar la evaluación, la base de datos cruza los siguientes factores para recomendar la mejor opción:

Método de Acceso
Táctil, mirada, pulsador, control oral...
Nivel de Fatiga
Esfuerzo físico requerido y resistencia muscular actual.
Entorno de Uso
Uso en cama, silla de ruedas (necesidad de anclajes) o exteriores.
Capacidad Sensorial
Evaluación visual (contraste. tamaño de fuente), auditiva y tecnológica.
Nivel Cognitivo
Comprensión para elegir el tipo de metodología de comunicación (Bliss, Minspeak, etc.).
Idioma y Alfabetización
Idioma nativo y nivel de lectoescritura preservado.
Plataforma
Windows, Android, iOS, Web o hardware propio.
Independencia
Uso autónomo o necesidad de ayuda para configuración.
Coste y Soporte
Presupuesto disponible y mantenimiento técnico requerido.
