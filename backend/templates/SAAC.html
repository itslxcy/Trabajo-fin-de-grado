Sistemas Aumentativos y Alternativos de Comunicación (SAAC)

Los SAAC son herramientas diseñadas para facilitar o sustituir el habla en personas con dificultades de comunicación oral. Su objetivo principal es garantizar que el usuario pueda expresar sus necesidades, sentimientos e ideas.

Definición conceptual

  • Sistema aumentativo: Apoya el habla residual para que la comunicación sea eficaz.
  • Sistema alternativo: Sustituye totalmente el habla cuando esta no es funcional.

Sistemas de símbolos y metodologías

El lenguaje se organiza mediante diferentes bibliotecas y métodos para optimizar la velocidad comunicativa:

Bibliotecas de símbolos

  • Widgit: Escritura inteligente con asignación automática de imágenes mientras se escribe.
  • ARASAAC / SPC: Pictogramas de alta claridad y uso universal.
  • Bliss / Benson: Sistemas para necesidades cognitivas o visuales específicas mediante representación con símbolos o alto contraste.
  • VSD: Escenas visuales basadas en fotos reales.

Metodologías clave

  • Minspeak: Sistema de iconos que combinandose entre ellos adquieren multi-significado (máxima velocidad).
  • Alfabética: Uso de teclados con predicción inteligente.
  • PODD: Organización basada en la intención pragmática (no se clasifica por temas sino por intenciones).

Clasificación de soluciones disponibles

Para una comunicación robusta en la ELA, se recomienda combinar un sistema de alta tecnología con uno de baja tecnología como respaldo.

Baja Tecnología

Soportes físicos y sistemas sin electrónica.

  • Tableros E-TRAN / MegaBEE: Comunicación visual directa.
  • SpeakBook: Cuadernos para mirada o puntero.
  • Paneles Físicos: Tableros alfabéticos y pictográficos.

Software y apps

Programas que gestionan el lenguaje y la voz.

  • Software avanzado: Grid 3, Verbo, TD Snap, ComuniQa.
  • Apps de mirada: Look to Speak, Tallk.
  • Comunicadores web: Cboard, LetMeTalk, Picto4Me.
  • Apps de voz: Asistente AAC, Speak4Me.

Hardware y periféricos

Dispositivos de acceso y complementos técnicos.

  • Seguimiento ocular: Eye trackers.
  • Punteros: Puntero láser manual y gafas con puntero (control cefálico).
  • Control de voz: Voice Access y Speech to Text.
  • Identidad: Bancos de voz y síntesis personalizada.
