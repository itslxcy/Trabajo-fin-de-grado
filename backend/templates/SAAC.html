Sistemas Aumentativos y Alternativos de Comunicación (SAAC)

Los SAAC son herramientas diseñadas para facilitar o sustituir el habla en personas con dificultades de comunicación oral. Su objetivo principal es garantizar que el usuario pueda expresar sus necesidades, sentimientos e ideas.

Definición conceptual

  • Sistema aumentativo: Apoya el habla residual para que la comunicación sea eficaz.
  • Sistema alternativo: Sustituye totalmente el habla cuando esta no es funcional.

Sistemas de símbolos y metodologías

El lenguaje se organiza mediante diferentes bibliotecas y métodos para optimizar la velocidad comunicativa:

Bibliotecas de símbolos

  • Widgit: Escritura inteligente con asignación automática de imágenes.
  • ARASAAC / SPC: Pictogramas de alta claridad y uso universal.
  • Bliss / Benson: Sistemas para necesidades cognitivas o visuales específicas.
  • VSD: Escenas visuales basadas en fotos reales.

Metodologías clave

  • Minspeak: Iconos combinados con multi-significado (máxima velocidad).
  • Alfabética: Uso de teclados con predicción inteligente.
  • PODD: Organización basada en la intención pragmática del usuario.

Clasificación de soluciones disponibles

Se recomienda combinar un sistema de alta tecnología con uno de baja tecnología como respaldo.

Baja Tecnología

Soportes físicos

  • Tableros E-TRAN: Láminas transparentes para leer la mirada.
  • SpeakBook: Cuadernos de comunicación rápida por mirada o puntero.
  • Paneles Físicos: Tableros de cartón con el alfabeto o pictogramas.

Software y Apps

Gestión del lenguaje

  • Software avanzado: Grid 3, Verbo o TD Snap para control total.
  • Apps de mirada: Look to Speak o Tallk mediante cámara frontal.
  • Comunicadores web: Cboard o LetMeTalk, accesibles y gratuitos.

Hardware

Dispositivos de acceso

  • Eye trackers: Cámaras de seguimiento ocular de alta precisión.
  • Punteros: Dispositivos de luz para manos o cabeza (gafas).
  • Bancos de voz: Procesos de grabación para preservar la identidad vocal.
